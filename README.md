# BTCBullF34ST
A zero-friction, browser-based arcade game for Bitcoiners and normies alike. No sign-ups. No ads. No tracking. No wallet connection. Just fun.
What Is This?
BTC Bull Run is a one-file HTML5 game where you play as a determined Bitcoin bull catching falling sats (satoshis) while a live market chart pumps and dumps behind you. Miss too many coins and the chart crashes — game over.
Built for:
Bitcoiners who want a quick dopamine hit between price checks
Normies curious about crypto culture but intimidated by wallets and jargon
Anyone who remembers the simple joy of Flash games and wants that back
How to Play
Open the file. Double-click index.html. That's it.
Pick your difficulty:
🌱 Easy — 3 lives, slower coins. Good for learning.
⚡ Normal — 2 lives, standard speed. The real experience.
🔥 Hard — 1 life, fast and brutal. Laser-eye territory.
Move. Mouse, touch, or arrow keys. Catch the ₿ coins.
Don't miss. If a coin hits the bottom, the chart crashes and you lose a life.
Build combos. Consecutive catches multiply your score. Hit 10x combo for laser eyes.
Level up. Every 500 sats unlocks a new visual theme.

Technical Details
Engine: Vanilla JavaScript + HTML5 Canvas
Rendering: Dual canvas (background chart + foreground game) for performance
Storage: localStorage for leaderboard persistence
Size: ~15KB minified. Loads instantly.
